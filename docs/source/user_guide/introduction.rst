Introduction
============

WildIntel Tools is a command-line utility suite designed to manage, validate, and prepare camera trap data collected
within the **WildINTEL project**.

It ensures that collections and deployments follow standardized naming conventions, validates metadata consistency, and
prepares datasets for import into **Trapper**, the data management platform used in WildINTEL.

Main Features
--------------

- **Check collections** – Validate collection and deployment folder names.
- **Check deployments** – Verify timestamp order and deployment metadata.
- **Prepare for Trapper** – Flatten and enrich datasets for Trapper import.
- **Configuration manager** – Easily initialize, edit, and switch project settings.
- **Helper tools** – Test connections and perform utility checks.
